# Evaluacion-Markdown-Ahinoa-Vinueza

## Proceso de configuración 
1. Descargar GitHub Desktop en mi computadora 
2. Abrir la aplicación GitHub Desktop
3. Iniciar sesión con mi cuenta
4. Crear un nuevo repositorio
5. Escribir el nombre del repositorio
6. Crear el repositorio
7. Publicar el repositorio en GitHub 
8. Verificar que el repositorio sea público
9. Revisar en la página de GitHub que el repositorio este visible
